Module#spacename
spacename()

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Returns the name of module's container module.

  module Example
    class Demo
    end
  end

  Demo.name         #=&gt; &quot;Example::Demo&quot;
  Demo.spacename    #=&gt; &quot;Example&quot;
This used to be called dirname.

See also Module#basename.
