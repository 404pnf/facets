Module#alias_module_function
alias_module_function(new, old)

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Alias a module function so that the alias is also a module function. The typical #alias_method does not do this.

  module Demo
    module_function
    def hello
      &quot;Hello&quot;
    end
  end

  Demo.hello    #=&gt; Hello

  module Demo
    alias_module_function( :hi , :hello )
  end

  Demo.hi       #=&gt; Hello
