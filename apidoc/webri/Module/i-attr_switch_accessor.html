Module#attr_switch_accessor
attr_switch_accessor(*args)

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Create a toggle attribute. This creates two methods for each given name. One is a form of tester and the other is used to toggle the value.

  attr_accessor! :a
is equivalent to

  def a?
    @a
  end

  def a!(value=true)
    @a = value
    self
  end
