From /home/trans/Documents/programs/current/facets/doc/ri/UnboundMethod/cdesc-UnboundMethod.yaml

Return the name of the method.

Be aware that in ruby 1.9 UnboundMethod#name is defined already, but it returns a Symbol not a String.

  class X
    def foo; end
  end

  meth = X.instance_method(:foo)

  meth.name  #=&gt; &quot;foo&quot;
