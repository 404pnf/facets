Module#redefine_method
redefine_method(sym, aka=nil, &blk)

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Creates a new method for a pre-existing method.

If aka is given, then the method being redefined will first be aliased to this name.

  class Greeter
    def hello ; &quot;Hello&quot; ; end
  end

  Greeter.new.hello   #=&gt; &quot;Hello&quot;

  class Greeter
    redefine_method( :hello, :hi ) do
      hi + &quot;, friend!&quot;
    end
  end

  Greeter.new.hello   #=&gt; &quot;Hello, friend!&quot;
