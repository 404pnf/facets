Format duration.

Identifiers

    %w -- Number of weeks
    %d -- Number of days
    %h -- Number of hours
    %m -- Number of minutes
    %s -- Number of seconds
    %t -- Total number of seconds
    %x -- Duration#to_s
    %% -- Literal `%' character
Example

    d = Duration.new(:weeks =&gt; 10, :days =&gt; 7)
    =&gt; #&lt;Duration: 11 weeks&gt;
    d.strftime(&quot;It's been %w weeks!&quot;)
    =&gt; &quot;It's been 11 weeks!&quot;
