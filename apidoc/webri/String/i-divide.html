divide( re )

From /home/trans/Documents/programs/current/facets/doc/ri/String/cdesc-String.yaml

Breaks a string up into an array based on a regular expression. Similar to scan, but includes the matches.

  s = &quot;&lt;p&gt;This<b>is</b>a test.&lt;/p&gt;&quot;
  s.divide( /&lt;.*?\&gt;/ )
produces

  [&quot;&lt;p&gt;This&quot;, &quot;<b>is&quot;, &quot;</b>a test.&quot;, &quot;&lt;/p&gt;&quot;]
