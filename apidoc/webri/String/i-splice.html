String#splice
splice(idx, sub=nil)

From /home/trans/Documents/programs/current/facets/doc/ri/String/cdesc-String.yaml

This is basically the same as #store, but it acts like slice! when given only one argument.

Essentlay #slice, but writes rather than reads.

  a = &quot;HELLO&quot;
  a.splice(&quot;X&quot;, 1)
  a                #=&gt; &quot;HXLLO&quot;

  a = &quot;HELLO&quot;
  a.splice(1)    #=&gt; &quot;E&quot;
  a              #=&gt; &quot;HLLO&quot;
