Module#prepend
prepend(aspect)

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Prepend an aspect module to a module. This only works at the module level.

  module X
    def x; &quot;x&quot;; end
  end

  module U
    def x; '{' + super + '}'; end
  end

  X.prepend U

  X.x  # =&gt; &quot;{x}&quot;
