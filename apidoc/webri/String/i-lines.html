String#lines
lines(&blk)

Returns an array of characters.

  &quot;abc\n123&quot;.lines  #=&gt; [&quot;abc&quot;,&quot;123&quot;]
Note, this is not 100% compatible with 1.8.7+ which returns an enumerator instead of an array.
