String#modulize
modulize()

From /home/trans/Documents/programs/current/facets/doc/ri/String/cdesc-String.yaml

Converts a string to module name representation.

This is essentially #camelcase. It also converts '/' to '::' which is useful for converting paths to namespaces.

Examples

  &quot;method_name&quot;.modulize    #=&gt; &quot;MethodName&quot;
  &quot;method/name&quot;.modulize    #=&gt; &quot;Method::Name&quot;
