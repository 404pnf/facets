Module#redirect_method
redirect_method( method_hash )

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Redirect methods to other methods. This simply defines methods by the name of a hash key which calls the method with the name of the hash's value.

  class Example
    redirect_method :hi =&gt; :hello, :hey =&gt; :hello
    def hello(name)
      puts &quot;Hello, #{name}.&quot;
    end
  end

  e = Example.new
  e.hello(&quot;Bob&quot;)    #=&gt; &quot;Hello, Bob.&quot;
  e.hi(&quot;Bob&quot;)       #=&gt; &quot;Hello, Bob.&quot;
  e.hey(&quot;Bob&quot;)      #=&gt; &quot;Hello, Bob.&quot;
The above class definition is equivalent to:

  class Example
    def hi(*args)
      hello(*args)
    end
    def hey(*args)
      hello(*args)
    end
    def hello
      puts &quot;Hello&quot;
    end
  end
