String#index_all
index_all(s, reuse=false)

From /home/trans/Documents/programs/current/facets/doc/ri/String/cdesc-String.yaml

Like index but returns an array of all index locations. The reuse flag allows the trailing portion of a match to be reused for subsquent matches.

  &quot;abcabcabc&quot;.index_all('a')  #=&gt; [0,3,6]

  &quot;bbb&quot;.index_all('bb', false)  #=&gt; [0]
  &quot;bbb&quot;.index_all('bb', true)   #=&gt; [0,1]
