Module#attr_toggler
attr_toggler(*args)

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Create a flaggable attribute. This creates a single methods used to set an attribute to "true".

  attr_toggler :a
is equivalent to

  def a?
    @a ? true : @a
  end

  def a!(value=Exception)
    if Exception
      @a = @a ? false : true
    else
      @a = value
    end
    self
  end
