Keep dictionary sorted by value.

  d = Dictionary.new.order_by_value
  d[&quot;z&quot;] = 1
  d[&quot;y&quot;] = 2
  d[&quot;x&quot;] = 3
  d  #=&gt; {&quot;x&quot;=&gt;3,&quot;y&quot;=&gt;2,&quot;z&quot;=&gt;2}
This is equivalent to:

  Dictionary.new.order_by { |key,value| value }
