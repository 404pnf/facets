Array#to_h_splat
to_h_splat()

From /home/trans/Documents/programs/current/facets/doc/ri/Array/cdesc-Array.yaml

This is equivalent to Hash[*array], but it will pad the array with a nil object if there are not an even number of elements.

  a = [:a,1,:b,2,:c]
  a.to_h_splat  #=&gt; { :a=&gt;1, :b=&gt;2, :c=&gt;nil }
