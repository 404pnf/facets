to_h (mode=nil)

Convert an Enumerable::Enumerator object into a hash. This is equivalent to Array#to_h. 

e1 = [[1,:a],[2,:b],[3,:c]].to_enum
e1.to_h #=> { 1=>:a, 2=>:b, 3=>:c }

e2 = [1,2,3,4,5].to_enum
e2.to_h  #=> {5=>nil, 1=>2, 3=>4}

e3 = [1,2,1,3,1,5].to_enum
e3.to_h #=> {1=>5}
