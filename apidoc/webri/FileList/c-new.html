FileList::new
FileList::new(*patterns) {|self if block_given?| ...}

From /home/trans/Documents/programs/current/facets/doc/ri/FileList/cdesc-FileList.yaml

Create a file list from the globbable patterns given. If you wish to perform multiple includes or excludes at object build time, use the "yield self" pattern.

Example:

  file_list = FileList.new['lib/*<b>/</b>.rb', 'test/test*.rb']

  pkg_files = FileList.new['lib/*<b>/</b>'] do |fl|
    fl.exclude(/\bCVS\b/)
  end
