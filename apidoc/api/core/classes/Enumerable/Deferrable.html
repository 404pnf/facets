Module Enumerable::Deferrable

  1. lib/core/facets/enumerable/defer.rb

Classes which include Enumerable::Deferrable will get versions of map, select etc. which return a Deferred object, so that they work horizontally without creating intermediate arrays.

    def skip(n)
      Enumerator::Deferred.new do |output|
        count = 0
        each do |*input|
          output.yield(*input) if count >= n
          count += 1
        end
      end
    end

      Enumerator::Deferred.new do |output|
