Automatically generate sorting defintions base on attribute fields.

  include SortOn(:a, :b)
is equivalent to including a module containing:

  def &lt;=&gt;(other)
    cmp = self.a &lt;=&gt; other.a; return cmp unless cmp == 0
    cmp = self.b &lt;=&gt; other.b; return cmp unless cmp == 0
    0
  end
