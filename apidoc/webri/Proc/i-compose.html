Proc#compose
compose(g)

From /home/trans/Documents/programs/current/facets/doc/ri/Proc/cdesc-Proc.yaml

Returns a new proc that is the functional composition of two procs, in order.

  a = lambda { |x| x + 4 }
  b = lambda { |y| y / 2 }

  a.compose(b).call(4)  #=&gt; 6
  b.compose(a).call(4)  #=&gt; 4
