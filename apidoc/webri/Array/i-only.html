Array#only
only()

From /home/trans/Documents/programs/current/facets/doc/ri/Array/cdesc-Array.yaml

Returns the only element in the array. Raises an IndexError if the array's size is not 1.

  [5].only      # -&gt; 5
  [1,2,3].only  # -&gt; IndexError
  [].only       # -&gt; IndexError
