Module#basename
basename()

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Returns the root name of the module/class.

  module Example
    class Demo
    end
  end

  Demo.name       #=&gt; &quot;Example::Demo&quot;
  Demo.basename   #=&gt; &quot;Demo&quot;
For anonymous modules this will provide a basename based on Module#inspect.

  m = Module.new
  m.inspect       #=&gt; &quot;#&lt;Module:0xb7bb0434&gt;&quot;
  m.basename      #=&gt; &quot;Module_0xb7bb0434&quot;
