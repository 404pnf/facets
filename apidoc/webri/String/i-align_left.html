String#align_left
align_left(n, sep="\n", c=' ')

From /home/trans/Documents/programs/current/facets/doc/ri/String/cdesc-String.yaml

Align a string to the left.

The default alignment separation is a new line ("\n"). This can be changed as can be the padding string which defaults to a single space (' ').

  s = &lt;&lt;-EOS
  This is a test
    and
    so on
  EOS

  puts s.align_left(20, &quot;\n&quot;, '.')
produces

  This is a test......
  and.................
  so on...............
