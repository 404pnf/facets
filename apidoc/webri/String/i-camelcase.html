String#camelcase
camelcase(first_letter=nil)

From /home/trans/Documents/programs/current/facets/doc/ri/String/cdesc-String.yaml

Converts a string to camelcase.

By default camelcase leaves the first character of the string as given. If first_letter is set to :lower or false, then #camelcase will produce lowerCamelCase. If it is set to :upper or true it will produce UpperCamelCase.

#camelcase also converts '/' to '::' which is useful for converting paths to namespaces.

Examples

  &quot;camel_case&quot;.camelcase             #=&gt; &quot;camelCase&quot;
  &quot;camel/case&quot;.camelcase(true)       #=&gt; &quot;Camel::Case&quot;
  &quot;Camel_case&quot;.camelcase(false)      #=&gt; &quot;camelCase&quot;
