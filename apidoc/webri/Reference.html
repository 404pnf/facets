Class: Reference < Object

Reference

Reference provides a way to access object indirectly. This allows for the object itself to be changed on the fly.

  a = &quot;HELLO&quot;
  b = ref(a)
  puts b    #=&gt; &quot;HELLO&quot;
  c = 10
  b.become(c)
  puts b    #=&gt; &quot;10&quot;
