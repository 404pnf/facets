Module#instance_method!
instance_method!(s)

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Access method as a singleton object and retain state.

  module K
    def hello
      puts &quot;Hello World!&quot;
    end
  end
  p K.instance_method!(:hello)   #=&gt; &lt;UnboundMethod: #hello&gt;
NOTE: This is limited to the scope of the current module/class.
