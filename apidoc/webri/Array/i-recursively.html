Array#recursively
recursively(&block) {|a| ...}

From /home/trans/Documents/programs/current/facets/doc/ri/Array/cdesc-Array.yaml

Apply a block to hash, and recursively apply that block to each subhash.

  arr = [&quot;a&quot;, [&quot;b&quot;, &quot;c&quot;, nil], nil]
  arr.recursively{|a| a.compact! }
  =&gt; [&quot;a&quot;, [&quot;b&quot;, &quot;c&quot;]]
