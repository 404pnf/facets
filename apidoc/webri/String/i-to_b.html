String#to_b
to_b()

From /home/trans/Documents/programs/current/facets/doc/ri/String/cdesc-String.yaml

Interpret common affirmative string meanings as true, otherwise false. Balnk sapce and case are ignored. The following strings that will return true:

  <tt>true</tt>,<tt>yes</tt>,<tt>on</tt>,<tt>t</tt>,<tt>1</tt>,<tt>y</tt>,<tt>==</tt>
Examples:

  &quot;true&quot;.to_b   #=&gt; true
  &quot;yes&quot;.to_b    #=&gt; true
  &quot;no&quot;.to_b     #=&gt; false
  &quot;123&quot;.to_b    #=&gt; false
