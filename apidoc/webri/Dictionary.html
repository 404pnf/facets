Class: Dictionary < Object

Dictionary

The Dictionary class is a Hash that preserves order. So it has some array-like extensions also. By defualt a Dictionary object preserves insertion order, but any order can be specified including alphabetical key order.

Usage

Just require this file and use Dictionary instead of Hash.

  # You can do simply
  hsh = Dictionary.new
  hsh['z'] = 1
  hsh['a'] = 2
  hsh['c'] = 3
  p hsh.keys     #=&gt; ['z','a','c']

  # or using Dictionary[] method
  hsh = Dictionary['z', 1, 'a', 2, 'c', 3]
  p hsh.keys     #=&gt; ['z','a','c']

  # but this don't preserve order
  hsh = Dictionary['z'=&gt;1, 'a'=&gt;2, 'c'=&gt;3]
  p hsh.keys     #=&gt; ['a','c','z']

  # Dictionary has useful extensions: push, pop and unshift
  p hsh.push('to_end', 15)       #=&gt; true, key added
  p hsh.push('to_end', 30)       #=&gt; false, already - nothing happen
  p hsh.unshift('to_begin', 50)  #=&gt; true, key added
  p hsh.unshift('to_begin', 60)  #=&gt; false, already - nothing happen
  p hsh.keys                     #=&gt; [&quot;to_begin&quot;, &quot;a&quot;, &quot;c&quot;, &quot;z&quot;, &quot;to_end&quot;]
  p hsh.pop                      #=&gt; [&quot;to_end&quot;, 15], if nothing remains, return nil
  p hsh.keys                     #=&gt; [&quot;to_begin&quot;, &quot;a&quot;, &quot;c&quot;, &quot;z&quot;]
  p hsh.shift                    #=&gt; [&quot;to_begin&quot;, 30], if nothing remains, return nil

Usage Notes

  • You can use #order_by to set internal sort order.

  • #<< takes a two element [k,v] array and inserts.

  • Use ::auto which creates Dictionay sub-entries as needed.

  • And ::alpha which creates a new Dictionary sorted by key.
