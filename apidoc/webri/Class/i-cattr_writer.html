Class#cattr_writer
cattr_writer(*syms)

From /home/trans/Documents/programs/current/facets/doc/ri/Class/cdesc-Class.yaml

Creates a class-variable attr_writer that can be accessed both on an instance and class level.

  class MyClass
    cattr_writer :a
    def a
      @@a
    end
  end

  MyClass.a = 10
  MyClass.a            #=&gt; 10
  MyClass.new.a = 29
  MyClass.a            #=&gt; 29
