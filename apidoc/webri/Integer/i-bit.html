Integer#bit
bit(bit)

From /home/trans/Documents/programs/current/facets/doc/ri/Integer/cdesc-Integer.yaml

Set a bit.

  0.bit!(4)  #=&gt; 8
Using an inverted bit will clear a bit.

  10.bit!(~3)      #=&gt; 2
  0xb0100.bit(~3)  #=&gt; 0
