The same as multiglob, but recusively includes directories.

  Dir.multiglob_r( 'folder' )
is equivalent to

  Dir.multiglob( 'folder', :recurse=&gt;true )
The effect of which is

  Dir.multiglob( 'folder', 'folder/*<b>/*</b>' )
