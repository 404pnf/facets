Learn Facets as you program, the Quick API makes it a snap. Or learn what others are doing with Facets on the Wiki. For special insights try the Super Highlights.
QUICK API

The Quick API provides a convenient interface mapped over the standard API documentation. If you are using Firefox it is especially convenient to drag-and-drop the "candy apples link" (left of this text) to a toolbar, then alter it's properties to open as a sidebar. If your browser doesn't support sidebars, or if you simply prefer it, you can use the mini api instead. This will offer the same interface, but in a small stand-alone window.
