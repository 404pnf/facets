Array#pad
pad(len, val=nil)

From /home/trans/Documents/programs/current/facets/doc/ri/Array/cdesc-Array.yaml

Pad an array with a given value upto a given length.

  [0,1,2].pad(6,&quot;a&quot;)  #=&gt; [0,1,2,&quot;a&quot;,&quot;a&quot;,&quot;a&quot;]
If length is a negative number padding will be added to the beginning of the array.

  [0,1,2].pad(-6,&quot;a&quot;)  #=&gt; [&quot;a&quot;,&quot;a&quot;,&quot;a&quot;,0,1,2]
