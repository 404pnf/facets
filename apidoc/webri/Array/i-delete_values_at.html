Array#delete_values_at
delete_values_at(*selectors)

From /home/trans/Documents/programs/current/facets/doc/ri/Array/cdesc-Array.yaml

Delete multiple values from array given indexes or index range.

  a = [1,2,3,4]
  a.delete_values_at(1,2)   #=&gt; [2,3]
  a                         #=&gt; [1,4]
  a = [1,2,3,4]
  a.delete_values_at(0..2)  #=&gt; [1,2,3]
  a                         #=&gt; [4]
