Enumerable::Enumerator#to_h
to_h(mode=nil)

From /home/trans/Documents/programs/current/facets/doc/ri/Enumerable/Enumerator/cdesc-Enumerator.yaml

Convert an Enumerable::Enumerator object into a hash. This is equivalent to Array#to_h.

  e1 = [[1,:a],[2,:b],[3,:c]].to_enum
  e1.to_h #=&gt; { 1=&gt;:a, 2=&gt;:b, 3=&gt;:c }

  e2 = [1,2,3,4,5].to_enum
  e2.to_h  #=&gt; {5=&gt;nil, 1=&gt;2, 3=&gt;4}

  e3 = [1,2,1,3,1,5].to_enum
  e3.to_h #=&gt; {1=&gt;5}
