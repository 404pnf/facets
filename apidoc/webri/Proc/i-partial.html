Proc#partial
partial(*args)

From /home/trans/Documents/programs/current/facets/doc/ri/Proc/cdesc-Proc.yaml

Convert a Proc object into new partial Proc object.

  a = proc { |a,b,c| a+b+c }
  b = a.partial(X,2,X)
  b[1,3] #=&gt; 6

  a = proc { |a,b,c| a+b+c }
  b = a.partial(__,2,__)
  b[1,3] #=&gt; 6
This method is similar to Proc#curry.
