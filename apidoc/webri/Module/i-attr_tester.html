Module#attr_tester
attr_tester(*args)

From /home/trans/Documents/programs/current/facets/doc/ri/Module/cdesc-Module.yaml

Create an tester attribute. This creates a single method used to test the attribute for truth.

  attr_tester :a
is equivalent to

  def a?
    @a ? true : @a
  end

(also known as attr_reader?)
