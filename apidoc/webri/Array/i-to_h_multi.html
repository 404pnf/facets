Array#to_h_multi
to_h_multi()

From /home/trans/Documents/programs/current/facets/doc/ri/Array/cdesc-Array.yaml

When a mixed or multi-element accociative array is used, the result is as follows:

  a = [ [:a,1,2], [:b,2], [:c], :d ]
  a.to_h  #=&gt; { :a=&gt;[1,2], :b=&gt;[2], :c=&gt;[], :d=&gt;[] }
If the first entry of the subelements is the same, then the values will be merged using #concat.

  a = [ [:a,1,2], [:a,3], [:a,4], [:a], :a ]
  a.to_h_multi  #=&gt; { :a=&gt;[1,2,3,4] }
