String#bracket
bracket(bra, ket=nil)

From /home/trans/Documents/programs/current/facets/doc/ri/String/cdesc-String.yaml

Return a new string embraced by given brakets. If only one bracket char is given it will be placed on either side.

  &quot;wrap me&quot;.bracket('{')        #=&gt; &quot;{wrap me}&quot;
  &quot;wrap me&quot;.bracket('--','!')   #=&gt; &quot;--wrap me!&quot;
