FileList#exclude
exclude(*patterns)

From /home/trans/Documents/programs/current/facets/doc/ri/FileList/cdesc-FileList.yaml

Register a list of file name patterns that should be excluded from the list. Patterns may be regular expressions, glob patterns or regular strings.

Note that glob patterns are expanded against the file system. If a file is explicitly added to a file list, but does not exist in the file system, then an glob pattern in the exclude list will not exclude the file.

Examples:

  FileList['a.c', 'b.c'].exclude(&quot;a.c&quot;) =&gt; ['b.c']
  FileList['a.c', 'b.c'].exclude(/^a/)  =&gt; ['b.c']
If "a.c" is a file, then ...

  FileList['a.c', 'b.c'].exclude(&quot;a.*&quot;) =&gt; ['b.c']
If "a.c" is not a file, then ...

  FileList['a.c', 'b.c'].exclude(&quot;a.*&quot;) =&gt; ['a.c', 'b.c']
