Indexable#index
index(obj=nil, &block) {|slice(i)| ...}

From /home/trans/Documents/programs/current/facets/doc/ri/Indexable/cdesc-Indexable.yaml

Returns the index of the first element equal to the given object or satisfying the block condition.

  [1,2,3,4].index{ |e| e == 3 }  #=&gt; 2
  [1,2,3,4].index{ |e| e &gt; 3 }   #=&gt; 3

(also known as index_of)
