DateTime#stamp
stamp(format=:default)

From /home/trans/Documents/programs/current/facets/doc/ri/DateTime/cdesc-DateTime.yaml

Convert to a formatted string. See Time::FORMAT for predefined formats.

This method is aliased to to_s.

Examples:

  datetime = DateTime.civil(2007, 12, 4, 0, 0, 0, 0)   # =&gt; Tue, 04 Dec 2007 00:00:00 +0000

  datetime.stamp(:db)            # =&gt; &quot;2007-12-04 00:00:00&quot;
  datetime.stamp(:db)            # =&gt; &quot;2007-12-04 00:00:00&quot;
  datetime.stamp(:number)        # =&gt; &quot;20071204000000&quot;
  datetime.stamp(:short)         # =&gt; &quot;04 Dec 00:00&quot;
  datetime.stamp(:long)          # =&gt; &quot;December 04, 2007 00:00&quot;
  datetime.stamp(:rfc822)        # =&gt; &quot;Tue, 04 Dec 2007 00:00:00 +0000&quot;

Adding your own datetime formats to stamp

DateTime formats are shared with Time. You can add your own to the Time::FORMAT hash. Use the format name as the hash key and a strftime string as the value. Eg.

  Time::FORMAT[:month_and_year] = &quot;%B %Y&quot;
